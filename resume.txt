Avancement du projet :
 Le jeu est lançable avec les noms des joueurs à taper dans la console
 Le premier ne peut etre placé qu'au centre et les joueurs choisisent ensuite librement l'emplacement
 La victoire avec alignement est gérée en vertical, horizontal et en diagonal(d'en haut a droite jusqu'en bas a droite)
 
Pour lancer le jeu:
 Le compiler en se placant dans le dossier ou les .java sont stockés avec javac *.java
 executer le Main avec java Main
 ensuite taper les deux noms de joueur dans la console quand ils sont demandés
